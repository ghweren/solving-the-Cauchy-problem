 The function that solves the Cauchy problem on a segment returns a table of values in the form of vector<vector<double>>. The function performs a calculation at a point using the "solvingMethod" solution method, then performs calculations for the same point using "estimationMethod", if the modulus of the absolute error value (the difference between two calculated values) is less than or equal to the maximum absolute error value (eps), then the calculation value is entered in the table, and the calculation continues for the next point, otherwise the step is divided into two and the calculation is repeated. Initially, the step is given by the formula (B - A) / 10. When moving to the next point, the possibility of reverse doubling of the step is calculated, if the value of the local absolute error divided by 2^k (the result of eps_order) is less than the absolute error, then the step is doubled. The step cannot be less than Hmin.

The function that solves the Cauchy problem on a segment returns a table of values in the form of vector<vector<double>>. The function performs a calculation at a point using the "solvingMethod" solution method, then performs calculations for the same point using "estimationMethod", if the modulus of the absolute error value (the difference between two calculated values) is less than or equal to the maximum absolute error value (eps), then the calculation value is entered in the table, and the calculation continues for the next point, otherwise the step is divided into two and the calculation is repeated. Initially, the step is given by the formula (B - A) / 10. When moving to the next point, the possibility of reverse doubling of the step is calculated, if the value of the local absolute error divided by 2^k (the result of eps_order) is less than the absolute error, then the step is doubled. The step cannot be less than Hmin.
